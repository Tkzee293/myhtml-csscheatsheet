Semantic Elements

Tag Name Description
<header> Header Represents the introductory content or a set of navigational links.
<main> Main Represents the dominant content of the body of a document.
<footer> Footer Represents the footer of a document or a section. It typically contains information like copyright notices, links to terms of service, contact information, or navigation links.
<section> Section Used to define a section of a document. It groups related content together, typically with a heading, and is often used to break up a webpage into distinct parts.
<div> Division A generic container for grouping and styling.
<nav> Navigation Contains navigation links.
<article> Article Represents a self-contained piece of content.
<aside> Aside Contains content that is tangentially related to the content around it.
